# MyCircle Radar App

Standalone Vue + Vite prototype inspired by Thoughtworks Technology Radar.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- Rings: Inner (Adopt), Middle (Trial), Outer (Assess)
- Quadrants: Knives, Stones, Songs
- Prototype is static and read-only for now
